Google Sheet Connection — 3 simple steps
Step 1: Open your Google Sheet "Phase I and II Man-Days" → File → Share → Publish to web → Select the sheet tab → Choose CSV → Click Publish. Copy the URL given.

Step 2: Paste that URL below. The dashboard will auto-read your sheet every time you open it.

Step 3 (optional): Set column numbers matching your sheet. Defaults match the data structure seen so far.

⚠️ No API key needed. No backend needed. The "Publish to web" URL is public-read — only people with the link can see the data. Learn more →
